---

## 一、项目概览

**定位**：三川田展厅智能估价获客工具（A阶段）  
**核心价值**：客户自助输入参数 → 秒出造价区间 → 业务员跟进转化

```
客户/业务员填写参数 → 概算引擎计算 → 结果展示 + PDF下载
                                      → 生成分享链接 → 客户留资 → 顾问跟进
```

### 3.4 概算引擎逻辑

```
总价 = (装修费 + 展陈费 + 灯光音响费 + 展项费) × 城市系数 × 工期系数 × (1 + 设计费率 + 税管费率)

- 装修费 = 面积 × 装修单价（按展厅类型×档次查表）
- 展陈费 = 面积 × 展除断单价
- 灯光音响费 = 面积 × 灯光音响单价
- 展项费 = 各选展项参考价之和
- 城市系数：一线1.0 / 二线0.85 / 三线0.7 / 四线0.6
- 工期系数：30天1.15 / 45天1.10 / 60天1.0 / 90天0.95 / 120天0.90
```
